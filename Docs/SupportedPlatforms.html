Supported Platforms

Getting Started ››

The Otter API was designed and developed as a cross-platform library and relies on minimal external dependencies to reduce porting effort to current and future platforms.  


The API comes with precompiled libraries for the following platforms:


Windows:


/lib/win/<vcver>/<platform>/OtterMT.lib

Multi-threaded Release, VC9, 32-bit

/lib/win/<vcver>/<platform>/OtterMD.lib

Multi-threaded DLL Release

/lib/win/<vcver>/<platform>/OtterMTD.lib

Multi-threaded Debug

/lib/win/<vcver>/<platform>/OtterMDD.lib

Multi-threaded DLL Debug

vcver : vc9 or vc10
platform : x86 or x64


iOS:


/lib/iOS/libOtter.a

Release

/lib/iOS/libOtterD.a

Debug


Android:


/lib/android/libOtter.a

Release

/lib/android/libOtterD.a

Debug
