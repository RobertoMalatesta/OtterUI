Masks

Editor ›› Tutorials ››

Masks are a way to selectively cull out pixels of a control during render time.  A mask, in the form of a texture, is provided and checked against during a control's render.  Any pixels of the control that coincide with the mask will be drawn to screen.  The rest will be culled out.


Creating a new Mask


    1. From the Controls panel, select the Mask control
    2. On the currently active view, click and drag to create the mask
    3. With the mask selected, click on the "Texture" property to open up the Texture Manager and select an appropriate texture



Using the Mask


    1. Select any control within the view
    2. Click on the "Mask" property and select any existing mask


The control will now render against the selected mask.


Notes


    • Masks apply recursively to all of the children of a control.
    • Child controls can override the parent's mask
    • The mask can be any texture, as long as it has an alpha channel.  Controls will only render when they coincide with mask pixels with alpha > 0.
