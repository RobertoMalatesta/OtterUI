Package Overview

Getting Started ››

Otter is packaged and delivered as a zip file, and contains all the necessary libraries, executables, documentation needed to start development.  


It is organized into largely into the following four areas:



API:


/API/inc        

Include directory

/API/lib/android

Android libraries

/API/lib/ios

iOS libraries

/API/lib/win32

Win32 libraries


Editor:


/Editor

Otter Editor installation executables (Win32 only)


Help:


/Help

Help Documentation


SampleApp:


/SampleApp

Sample Applications

/SampleApp/bin

Common assets, binary data, working folders, etc.

/SampleApp/bin/Assets

Sample App source assets

/SampleApp/bin/Data

Sample App exported assets

/SampleApp/src        

Common Sample Application source

/SampleApp/plaftorms/Android

Android Sample Application

/SampleApp/platforms/iOS

iOS Sample Application

/SampleApp/platforms/Win32

Win32 Sample Application
